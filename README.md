**随机点名程序 v2.17.0**

在程序目录下放一个"姓名及概率.txt"的文档,程序可以自动读取,或者在更多选项中选择任何.txt文档,文档的格式如下:
```text
姓名,概率
nameA,1
nameB,1
nameC,1
```
